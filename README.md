# pytracer
Attempt to build a ray tracing app abstracting various graphics libraries - mvc

Graphics Packages Integrated
- pygame

GUI Commands

|Key           |Description                                    |
|--------------|-----------------------------------------------|
|C             |Centre drawing / Reset                         |
|Arrow Keys    |Shift Up/Down/Left/Right                       |
|Plus/Minus    |Zoom In/Out                                    |
|Q/W           |Rotate X-Axis                                  |
|A/S           |Rotate Y-Axis                                  |
|Z/X           |Rotate Z-Axis                                  |
